# openclaw-contributor

Pi skill for contributing bug fixes to [openclaw/openclaw](https://github.com/openclaw/openclaw).

## Install

```bash
git clone https://github.com/arosstale/openclaw-contributor-skill ~/.pi/agent/skills/openclaw-contributor
```

## Prerequisites

- `gh` CLI authenticated
- Fork at `arosstale/openclaw`
- Clone at `/tmp/openclaw-fork` with remotes `origin` (upstream) and `fork` (ours)
